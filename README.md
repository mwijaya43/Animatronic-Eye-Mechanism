# Animatronic-Eye-Mechanism
This Project Makes Electronic Eye Mechanism With Servos and Joystick as a Controller

Components: 
- Arduino Uno (1)
- Servo (6)
- Servo Driver (1)
- Switch (1)
- Joystick (1)
- Adjustable Power Supply (1)
- Potentiometer (1)
- Resistor 10k Ohm (1)
